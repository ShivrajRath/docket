# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-15

### Added

- Initial release of DayDeck plugin
- Responsive section grid with drag-and-drop functionality
- Smart capture with inline tagging support
- Semantic tagging system with custom colors
- Task reminders with date/time pickers
- Waiting time counters for tracking elapsed time
- Archive system for completed tasks
- Multi-tag filtering capabilities
- Section customization (icons, colors, widths, descriptions)
- Tag management with visual indicators
- Desktop notifications for reminders
- Light and dark mode support
